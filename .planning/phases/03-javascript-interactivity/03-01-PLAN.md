**CRITICAL: Use existing CSS pattern `[data-animate]` / `is-visible`, NOT `.reveal` / `.reveal--visible`.** The CSS infrastructure already exists in `animations.css`. Per RESEARCH.md, D-12 class names are under Claude's Discretion and should align with existing CSS.

**1. Scroll-reveal setup (D-10, D-12, D-13):**
- Select reveal targets: `document.querySelectorAll('.service-card, .project-card, .process-step')`
- For each target: `el.setAttribute('data-animate', '')` — this activates the CSS hidden state (`opacity: 0; translateY(20px)`)
- Stagger delay (D-13): for each target, find its index among siblings in the same parent using `Array.from(el.parentElement.children).filter(c => c.matches('.service-card, .project-card, .process-step'))`. If sibling index > 0, set `el.setAttribute('data-delay', String(siblingIndex))` — this maps to existing CSS rules `[data-delay="1"]` (100ms), `[data-delay="2"]` (200ms), `[data-delay="3"]` (300ms)
- Create one `IntersectionObserver` with `{ threshold: 0.15 }` (D-12)
- In callback: if `entry.isIntersecting`, add `is-visible` class to `entry.target`, then `observer.unobserve(entry.target)` (reveal once only)
- Observe all targets

**2. Stats counter (D-11):**
- Select stat elements: `document.querySelectorAll('.stat-item__value[data-count-target]')`
- Create a SEPARATE `IntersectionObserver` on the `#services` section element (not on individual stat items) with `{ threshold: 0.3 }`
- When `#services` enters viewport (`entry.isIntersecting`):
  - `observer.unobserve(entry.target)` immediately (fire once — Pitfall 5)
  - For each stat element: read `parseInt(el.dataset.countTarget, 10)` to get target number
  - Call `animateCount(el, target, 1500)` for each
- `animateCount(el, target, duration)` function:
  - Uses `requestAnimationFrame` loop
  - `const start = performance.now()`
  - On each frame: `const progress = Math.min((now - start) / duration, 1)`
  - Ease-out curve: `const eased = 1 - Math.pow(1 - progress, 2)`
  - Set `el.textContent = Math.floor(eased * target) + '+'` (restores "+" suffix per D-11)
  - Continue loop while `progress < 1`

**3. No DOMContentLoaded needed** — scripts are `defer`.
